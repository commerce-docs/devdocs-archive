    Staging (Adobe Commerce only)

    Staging functionality enables you to create scheduled campaigns, manage campaigns on dashboard and manipulate each future store object update. Moreover, you can preview a storefront during the future campaign.

    Information on how the Staging functionality works and how to use the various features, can be found in the Content Staging topic of the User Guide.

    The functionality is distributed between staging modules. The Staging module is a framework for the staging functionality, and other staging modules are extensions for the existing modules. You can read about each staging module in a corresponding topic:
